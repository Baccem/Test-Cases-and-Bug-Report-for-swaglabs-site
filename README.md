# 🧪 Swag Labs - Test Cases & Bug Reports

This repository contains **manual testing** for the Swag Labs (SauceDemo) web application.  
It demonstrates a structured approach to testing, including **test case design**, **bug reporting**, and **test coverage**.

---

## 📌 Project Overview

The purpose of this project is to validate the core functionalities of the Swag Labs website, such as:

- User Authentication (Login)
- Product Listing
- Cart Management
- Checkout Process
